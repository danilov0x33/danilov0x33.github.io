# Операционные системы для ПК

## Windows

### Описание
Сомневаюсь, что об этой операционной системе вам не известно, но если коротко,
то это самая популярная и распространенная ОС для ПК для "бытового" пользователя вот уже долгое время.
Отсюда вытекают все преимущества этой системы.

### Преимущества/недостатки

#### Преимущества
1. Почти все можно сделать/настроить взаимодействуя с интерфейсом (даже настроить Windows Server)
2. Если любите частенько поиграть в свежие игры, то это ваш выбор
3. **Огромное** кол-во ПО написана/импортировано под эту систему
4. Возможность запуска Linux ядра прямо из "коробки"
5. **Огромная** база драйверов
6. Большое сообщество. Скорее всего проблемы с которым встречались уже кем-то было решено и это решение есть в интернете
7. Удаленное управление рабочим столом (RDP) из "коробки"
8. Очень простое и удобное управление Windows станциями в корпоративном сегменте посредством Windows Server (Ms AD)

#### Недостатки
1. Платная. На текущий момент это разовая оплата конкретной версии, но в будущем планируется подписочная модель (для ПК)
2. Ресурсозатратная система по сравнению с другими ОС
3. Принудительные обновления. В сети есть инструкции как это отключить, но все же
4. Обновления не всегда стабильные
5. **Огромное** кол-во вирусов и всех видов


## Mac

### Описание
Mac OS - это ОС из Unix семейства. Полностью проприетарная и закрывая система.

### Преимущества/недостатки

#### Преимущества
1. Почти все можно сделать/настроить взаимодействуя с интерфейсом
2. Отличная оптимизация ПО на официальных устройствах с MacOS
3. Целая экосистема. Если у вас вся продукция Apple, то все преимущества раскроются именно при использовании MacOS и наоборот
4. По сравнению с Windows кол-во вирусов меньше, хотя они все же есть
5. Больше "заточен" под графические и мультимедийные функции

#### Недостатки
1. ОС "заточена" под конкретное железо и официально MacOS поставить на другое железо нельзя
2. Иногда придется залезать в терминал
3. Устройства с этой ОС дороже, чем сборка ПК с похожим железом
4. Не распространен для серверов. Есть macOS Server, но по популярности сильно уступает другим ОС из этого списка
5. Кол-во видео игр поддерживаемых данную ОС сильно меньше чем для Windows
6. Далеко не всё ПО которое есть под Windows, есть в MacOS. В основном это касается какого-то специфичного ПО, но все же стоит учитывать


## Linux

### Описание
Linux - это целое семейство ОС. Я бы даже сказал, что целы зоопарк различных версий/сборок. Linux является ядром,
поэтому есть множество дистрибутивов (набор файлов, необходимых для установки ПО), выполняющих конкретные задачи.

### Преимущества/недостатки

#### Преимущества
1. Огромное кол-во дистрибутивов под разные задачи
2. Не ресурсоемкая и производительная система
3. Идеальна для серверов и контейнеризации (например для docker образов)
4. Большой простор для кастомизации
5. Бесплатная. Однако есть и платные дистрибутивы
6. По сравнению с Windows кол-во вирусов меньше, хотя они все же есть
7. Можно запускать Windows приложения через Wine, но не всё ПО будет работать корректно

#### Недостатки
1. Тут твой друг - это терминал. И без него никуда
2. Относительно сложная система для рядового пользователя
3. Кол-во видео игр поддерживаемых данную ОС сильно меньше чем для Windows
4. Далеко не всё ПО которое есть под Windows, есть в Linux, а аналоги предоставляют меньший функционал (Gimp VS Photoshop)

## Личный опыт
Я использую Windows как основную систему т.к. иногда поиграть возникает желание. А переключатся между ОС - оказалось дико не удобно.
Поэтому через виртуализацию могу запустить любую ОС т.к. linux подобные системы менее ресурсозатратные + теперь из коробки в Windows есть Linux.

## Вывод
Каждая система по своему хороша для тех или иных целей. Я бы их использовал так:
* Если нужна игровая платформа или есть старый и специфичный софт (такие встречаются в гос. предприятиях), или у вас мощный ПК, то ваш выбор - это Windows
* Если вы используете Apple продукцию, работаете с 3D графикой/видеомонтажом и готовы покупать Apple устройства, то ваш выбор - это MacOS
* Если всего остального есть Linux :)
